# Tunisian Accounting

This module by itself it does nothing, it just a wrapper module that add the possibility to install another modules from the settings

**Table of Contents**

- Features & Limitations
- Configuration
- Usage
- Dependencies

---

## Features

- Include payment methods like checks, banknotes, cash and bank transfer.
- Include payment replacement to set a payment as unpaid and add the possibility to replace it.
- Include tax stamp to be added automatically to invoices and bills.

---

## Configuration

1. Go to settings.
2. Tunisian Accounting Section.
3. check your desired modules to install.
