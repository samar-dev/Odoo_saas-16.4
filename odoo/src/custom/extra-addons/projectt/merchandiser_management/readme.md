## Features

- Create assortment from partners with other address.
- Affect products to assortments.
- Add assortment to planning slots.
- Create merchandiser planning from resources planning slot.
- Add menu-item for merchandiser management.

---

## Configuration

- For merchandiser management users, add them to the Merchandise/User group.
